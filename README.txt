COMP90049 Knowledge Technologies - Project1

This README will introduces how does program work and some sources of the codes.

EditDistance.java: This section of code calculates the edit distance between two strings.

JaroWinklerDistance.java: This part of the code is used to calculate the Jaro-Winkler Similarity of the two strings.

Detect.java: It is main body of program. First it reads the data of three text files and stores them in the array list. After preprocessing the data, the above codes are used to filter the data aiming to obtain final results.

The Levenshtein Distance has been sourced from: https://blog.csdn.net/zzq900503/article/details/36901149

The Jaro-Winkler Distance has been sourced from: https://blog.csdn.net/a553181867/article/details/89057661
